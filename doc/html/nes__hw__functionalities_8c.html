◆ wait_until_nmi()

void wait_until_nmi ( void  )

This function waits until NMI interrupt is triggered during VBlank (see aöso .NMI section in reset.s). The difference to function wait_Vblank() is, that this function checks, if especially NMI is reached, not VBlank. Therefore this function should be used to frequently update graphics when the screen is on.

Use wait_Vblank() when you return from an disabled screen using ppu_turn_all_on() .
